# README

## アプリ名： ReviewInEnglish
アプリURL: [https://review-in-english.herokuapp.com/users/sign_in](https://review-in-english.herokuapp.com/users/sign_in)

## 概要
英語で書く映画のレビューアプリ。  

## コンセプト
英語学習者をターゲットにしている。  
英語の上達にはアウトプットがかかせない。  
日常会話を上達のために映画のレビューを英語で書くとかなり効果的。  
SNSライクにレビューを添削しあえる。
